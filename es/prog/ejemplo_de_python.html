Manejo simple de archivos en Phyton

21/11/2020

Este es un ejemplo sacado de la página SoloLearn, es un problema denominado "Títulos de libros", el cual, se redacta así:
"Se te ha pedido realizar un programa especial de categorización de libros, el cual asigna a cada libro un código especial basado en su título. Por ejemplo, para el libro "Harry Potter", el código debe ser: H12, donde el número se refiere al de caracteres que contiene el título (incluyendo el espacio)".
"Se te ha proveido un archivo denominado books.txt, el cual include los títulos de libros, cada uno en una línea separada."
Por ejemplo, si el archivo contiene:
-) Algún libro
-) Otro libro
el programa debe arrojar como resultado:
-) A11
-) A10


El algoritmo queda de la siguiente manera:


1 2 3 4
file = open("books.txt", "r")
for line in file:
    print (line[0] + str(len(line)-1))
file.close()
